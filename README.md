# mms-trashcan

Trashcan System for VORP Core

# Features
 
- Place Trashcans Around your Map and Town
- Every Trashcan can be Searched Once then after The Reset Again
- Reward System Money or Items or Both
- Trashcans can be Filled and Emptied 
- Webhook Suppport

# installation 

- Add `mms_trashcan` to your resources
- Add `ensure mms_trashcan` to your `server.cfg`

# Required

- vorp_core
- vorp_inventory
